# 版本规划路线图 / Version Roadmap

## 项目概述 / Project Overview

**Website Block Extension** 是一款Chrome浏览器插件，旨在通过时间限制和多层验证机制帮助用户提升专注力，减少对分心网站的依赖。

---

## 版本历史与规划 / Version History & Planning

### ✅ v1.0.0 - 核心功能版本 (开发中 / In Development)

**发布时间 / Release Date**: 预计 2024 Q1

**开发状态 / Status**: 🚧 开发中

**核心目标 / Core Goals**:
建立插件的核心功能框架，实现基础的时间限制、三状态管理和多层验证重启机制。

**主要功能 / Key Features**:

#### 1. 基础时间限制系统
- ✅ 为每个网站设置每日使用时间限制
- ✅ 实时倒计时显示
- ✅ 跨tab时间同步
- ✅ 自动状态管理

#### 2. 固定时间锁定
- ✅ 支持多个时间段配置
- ✅ 跨午夜时间段支持（如 22:00-06:00）
- ✅ 两种限制模式：特定网站 / 所有HTTP/HTTPS
- ✅ 时间段标签管理

#### 3. 三状态管理系统
- ✅ Active（可使用）：正常计时
- ✅ Pending（待锁定）：30秒缓冲期
- ✅ Locked（已锁定）：完全阻止访问

#### 4. 30秒待锁定缓冲机制
- ✅ 全屏毛玻璃遮罩效果
- ✅ 动态模糊度变化
- ✅ 紧急使用功能（+10分钟）
- ✅ 立即锁定按钮

#### 5. 多层验证重启系统
- ✅ 第一层：10秒冷静倒计时
- ✅ 第二层：120个单词输入验证
- ✅ 紧急重启（每日一次全局限制）
- ✅ 礼花效果激励
- ✅ 重启历史记录

#### 6. UI/UX系统
- ✅ macOS风格毛玻璃效果
- ✅ 右下角可拖拽悬浮窗
- ✅ Options Page（4个Tab）
- ✅ Popup快速概览
- ✅ 激励性阻止页面

#### 7. 数据管理
- ✅ 本地存储
- ✅ 配置导入/导出
- ✅ 每日自动重置

**技术栈 / Tech Stack**:
- Plasmo Framework
- React 18
- TypeScript
- Tailwind CSS
- Day.js
- canvas-confetti
- Recharts
- Lucide React

## 功能优先级说明 / Priority Levels

### P0 - 高优先级 (High Priority)
- 对用户体验有重大提升
- 核心功能的重要补充
- 市场竞争力的关键因素

### P1 - 中优先级 (Medium Priority)
- 重要但非紧急的功能
- 增强产品竞争力
- 用户需求较高

### P2 - 低优先级 (Low Priority)
- 锦上添花的功能
- 可以等待合适时机实现
- 用户需求相对较低

## 版本迭代原则 / Version Iteration Principles

### 1. 用户驱动
- 优先实现用户反馈最多的功能
- 定期收集用户意见
- 快速响应用户需求

### 2. 渐进增强
- 每个版本专注核心目标
- 保持功能稳定性
- 避免功能膨胀

### 3. 质量优先
- 充分测试后再发布
- 重视用户体验
- 性能优化持续进行

### 4. 文档同步
- 每个版本都有完整文档
- 保持中英双语文档同步
- 及时更新changelog
