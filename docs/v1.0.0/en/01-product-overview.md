## 1. Product Introduction

### 1.1 Product Name
**Website Block Extension** - Time Management Extension for Chrome

### 1.2 Product Positioning
A Chrome browser extension that helps users improve focus and reduce dependence on distracting websites through multi-layer verification mechanisms and motivational design.

### 1.3 Core Philosophy
**"Gentle Yet Firm Self-Discipline Assistant"**

- **Gentle**: 30-second buffer, motivational quotes, confetti effects
- **Firm**: Multi-layer verification (10-second countdown + 120-word input)
- **Assistant**: A partner helping users build good habits

---

## 2. Target Users

### 2.1 Primary User Groups

#### Students
- **Pain Point**: Easily distracted by social media and video sites
- **Need**: Control entertainment website access during study time

#### Professionals
- **Pain Point**: Work interrupted by news and shopping sites
- **Need**: Improve work efficiency

#### Freelancers
- **Pain Point**: Easy to lose focus when working from home
- **Need**: Strict time management

---

## 3. Core Value

### 3.1 Value for Users

#### Improve Focus
- Reduce distractions
- Build focused work/study habits
- Save time

#### Build Self-Discipline
- Strengthen self-control through multi-layer verification
- Positive reinforcement through confetti effects
- Self-awareness through usage statistics

---

## 4. Key Features

### 4.1 Four Core Modules

#### Module 1: Control Panel System
- **Popup**: Quick usage overview
- **Options Page**: Complete configuration (4 Tabs)

#### Module 2: Content Injection System
- **Timer Overlay**: macOS glassmorphism, draggable

#### Module 3: Time Management & Locking System
- **Three-State Management**: Active / Pending / Locked
- **30-Second Buffer**: Full-screen glassmorphism overlay

#### Module 4: Block Page & Restart System
- **Motivational Block Page**: Quotes, statistics
- **Multi-Layer Restart**: 10 seconds + 120 words
