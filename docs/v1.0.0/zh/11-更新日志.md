## [1.0.0] - TBD

### 新增功能

#### 控制后台系统
- Popup页面显示今日使用情况
- Options Page（4个Tab）
- 网址清单管理
- 固定时间锁定配置
- 数据导入/导出

#### 内容注入系统
- 倒计时悬浮窗（macOS毛玻璃效果）
- 展开/收起两种状态
- 支持拖拽

#### 时间管理系统
- 三状态管理（Active/Pending/Locked）
- 30秒待锁定缓冲
- 跨Tab时间同步
- 每日自动重置

#### 重启系统
- 多层验证重启机制
- 10秒倒计时
- 120个单词输入验证
- 紧急重启（每日一次）
- 礼花效果

### 技术实现
- Plasmo Framework
- React 18
- TypeScript
- Tailwind CSS
