# 功能清单 / Feature Checklist

## 📖 使用说明 / Instructions

本文档用于跟踪Website Block Extension的所有功能开发进度。

### 状态标记 / Status Indicators
- ✅ = 已完成 (Completed)
- 🚧 = 开发中 (In Progress)
- 📋 = 待开发 (Planned)
- ❌ = 已取消 (Cancelled)

### 更新规则 / Update Rules
1. **开始开发**：将状态从 📋 改为 🚧
2. **完成功能**：将状态从 🚧 改为 ✅，填写完成日期
3. **取消功能**：将状态改为 ❌，说明原因
4. **添加新功能**：在对应版本下添加新行
